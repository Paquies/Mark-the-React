## Adding MDX to Webpack/Bundler
<br/>
To use MDX in your project, you need to configure your bundler (like Webpack) to handle <span className='highlight'>{`.mdx`}</span> files. Here are the steps to configure Webpack for MDX:

1. **Install necessary packages**:
  <CopyableCode language="bash">
    {`npm install @mdx-js/loader @mdx-js/react @mdx-js/runtime`}
</CopyableCode>
2. **Update Webpack configuration**:
<CopyableCode language='jsx'>
   { ` module.exports = {
      module: {
         rules: [
            {
              test: /\.mdx$/,
              use: [
                 'babel-loader',
                 '@mdx-js/loader'
              ]
            }
         ]
      }
    };`}
</CopyableCode>
This configuration tells Webpack to use <span className='highlight'>{`babel-loader`}</span> and <span className='highlight'>{`@mdx-js/loader`}</span> for <span className='highlight'>{`.mdx`}</span> files.

<br/>
## Setting up MDX Provider
<br/>
The <span className='highlight'>MDXProvider</span> component allows you to customize how MDX content is rendered. Here's how to set it up:

1. **Import MDXProvider**:

  <CopyableCode language='jsx'>
    {`import { MDXProvider } from '@mdx-js/react';`}
  </CopyableCode>

2. **Define custom components**:

<CopyableCode language="jsx">
    {`const components = {
      h1: (props) => <h1 style={{ color: 'tomato' }} {...props} />,
      // Add more custom components as needed
    };`}
</CopyableCode>

3. **Wrap your application with MDXProvider**:

  <CopyableCode language="jsx">
    {`function App() {
      return (
         <MDXProvider components={components}>
            <YourMDXContent />
         </MDXProvider>
      );
    }`}
</CopyableCode>

This setup allows you to customize the rendering of MDX content using your own React components.

<br/>
## Using Custom Components
<br/>
You can define and use custom React components within your MDX files. Here's how:

1. **Create a custom component**:

<CopyableCode language="jsx"> 
    {`const CustomButton = ({ children }) => (
      <button style={{ backgroundColor: 'blue', color: 'white' }}>
         {children}
      </button>
    );`}
</CopyableCode>
2. **Import and use the custom component in your MDX file**:

<CopyableCode language='markdown'>
    {`import CustomButton from './path/to/CustomButton';

    # Using Custom Components

    <CustomButton>Click Me!</CustomButton>`}
</CopyableCode>
This allows you to enhance your MDX content with interactive and reusable React components.
