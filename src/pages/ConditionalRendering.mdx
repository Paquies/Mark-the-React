### Conditional Rendering in React

Conditional rendering is a technique in React that allows you to render different components or elements based on certain conditions. This is useful when you want to display content conditionally based on user input, state changes, or other factors.

In this guide, we will explore different ways to implement conditional rendering in React using JavaScript expressions, if statements, and the ternary operator.

Let's start with a simple example:

```jsx
import React, { useState } from 'react';

const ConditionalRendering = () => {
  const [isLoggedIn, setIsLoggedIn] = useState(false);

  return (
    <div>
      {isLoggedIn ? (
        <h2>Welcome, User!</h2>
      ) : (
        <button onClick={() => setIsLoggedIn(true)}>Log In</button>
      )}
    </div>
  );
};

export default ConditionalRendering;
```

In the example above, we have a `ConditionalRendering` component that renders a welcome message if the user is logged in, or a login button if the user is not logged in. The `isLoggedIn` state is initialized to `false` using the `useState` hook, and the state is updated when the user clicks the login button.

This is a simple example of conditional rendering using the ternary operator. You can also use JavaScript expressions and if statements to conditionally render components based on different conditions.

### Using If Statements

You can use if statements to conditionally render components. Here's an example:

```jsx
import React, { useState } from 'react';

const ConditionalRendering = () => {
  const [isLoggedIn, setIsLoggedIn] = useState(false);

  if (isLoggedIn) {
    return <h2>Welcome, User!</h2>;
  } else {
    return <button onClick={() => setIsLoggedIn(true)}>Log In</button>;
  }
};


export default ConditionalRendering;
```
